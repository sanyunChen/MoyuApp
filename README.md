# 🐟 摸鱼办仪表盘 - WPF桌面版

基于 [moyu-office-tracker](https://github.com/65316571/moyu-office-tracker) 创意开发的现代化Windows桌面应用程序，使用WPF和Material Design设计，专为职场人士打造的高效摸鱼神器！

## ✨ 项目特色

### 🎯 创意来源

本项目灵感来源于 [moyu-office-tracker](https://github.com/65316571/moyu-office-tracker) 的纯前端HTML版本，在保留核心功能的基础上，使用WPF技术栈重新开发，带来了更丰富的交互体验和现代化界面设计。

### 🚀 核心功能

#### 📊 智能数据展示

- **实时时钟**: 右上角时间每秒跳动，精确显示当前时间
- **今日进度**: 基于自定义工作时间计算当日工作完成百分比
- **本周进度**: 显示本周工作进度，帮助规划工作安排
- **周末倒计时**: 精确到秒的周末倒计时，摸鱼动力满满
- **发薪倒计时**: 自定义发薪日，提前规划财务安排
- **退休倒计时**: 基于性别和入职日期计算退休进度（男60岁/女55岁）
- **节假日倒计时**: 自动显示即将到来的节假日信息
- **摸鱼语录**: 智能语录系统，根据时间和场景推送不同内容

#### 🎨 现代化界面设计

- **Material Design**: 采用Google Material Design设计规范
- **气泡动画**: 仿HTML版本的动态气泡背景效果
- **响应式布局**: 自适应窗口大小调整
- **平滑动画**: WPF原生动画，流畅自然

#### 🔧 强大的交互功能

- **实时编辑**: 双击任意模块即可编辑内容
- **自动保存**: 配置更改自动保存到本地
- **数据备份**: 支持配置备份和恢复功能
- **导入导出**: JSON格式配置导入导出
- **系统托盘**: 支持最小化到系统托盘

#### ⚙️ 个性化配置

- **工作时间**: 自定义上班/下班时间
- **个人信息**: 设置入职日期和性别（用于退休计算）
- **发薪设置**: 灵活配置发薪日和工作日范围
- **语录设置**: 自定义积极语录显示时间段
- **自定义倒计时**: 添加个人重要事件倒计时
- **刷新间隔**: 自定义界面刷新频率

## 🏗️ 技术架构

### 核心技术栈

- **框架**: .NET 8.0 + WPF (Windows Presentation Foundation)
- **UI库**: Material Design In XAML Toolkit
- **数据存储**: JSON序列化 + 本地文件存储
- **架构模式**: MVVM (Model-View-ViewModel)
- **定时器**: DispatcherTimer (UI线程安全)
- **动画**: WPF原生动画系统

## 🚀 快速开始

### 环境要求

- **操作系统**: Windows 10/11 (64位)
- **运行时**: .NET 8.0 Runtime 或更高版本
- **开发环境**: Visual Studio 2022 或更高版本 (可选)

### 首次使用配置

1. **启动程序**: 双击 `摸鱼办` 运行
2. **打开设置**: 点击右下角齿轮图标 ⚙️
3. **基础配置**:
   - 设置您的【入职日期】和【性别】（退休计算必需）
   - 配置【上班时间】和【下班时间】
   - 设置【发薪日】（1-31号）
4. **个性化设置**:
   - 调整【刷新间隔】（默认60秒）
   - 配置【积极语录时间段】
5. **保存配置**: 点击保存按钮，配置立即生效

## 🎯 使用指南

### 基础操作

- **查看信息**: 程序启动后自动显示所有倒计时和进度信息
- **编辑内容**: 双击任意模块的文本即可进入编辑模式
- **保存编辑**: 按Enter键或点击保存按钮确认修改
- **取消编辑**: 按ESC键或点击取消按钮放弃修改

### 高级功能

- **自定义倒计时**: 在设置中添加个人重要事件
- **数据备份**: 定期备份配置文件到安全位置
- **导入导出**: 迁移设置到其他设备
- **系统托盘**: 最小化到托盘，随时查看时间

### 数据管理

配置文件保存在：

```
%APPDATA%\摸鱼办\config.json
```

备份文件位置：

```
%APPDATA%\摸鱼办\Backups\
```
